## యంత్రాలు ఎలా చూస్తాయి? 👀

మీరు ఒక ఫోటోను చూసి క్షణంలో కుక్కను, కారును లేదా మీ ప్రాణ స్నేహితుడి ముఖాన్ని గుర్తిస్తారు. మీ మెదడు దీన్ని అలవోకగా చేస్తుంది — కానీ కంప్యూటర్‌కు "చూడడం" అనేది చాలా సంక్లిష్టమైన పని. **కంప్యూటర్ విజన్** అనేది ప్రపంచంలోని దృశ్య సమాచారాన్ని — చిత్రాలు, వీడియోలు, లైవ్ కెమెరా ఫీడ్‌లను — అర్థం చేసుకునే మరియు విశ్లేషించే సామర్థ్యాన్ని యంత్రాలకు అందించే AI రంగం.

## పిక్సెల్స్ నుండి ఫీచర్‌ల వరకు 🔍

కంప్యూటర్‌కు చిత్రం అంటే కేవలం సంఖ్యల గ్రిడ్ మాత్రమే.

### పిక్సెల్ అంటే ఏమిటి
- **గ్రేస్కేల్** చిత్రం ఒక 2D గ్రిడ్, ఇందులో ప్రతి సెల్‌లో 0 (నలుపు) నుండి 255 (తెలుపు) వరకు ఒక విలువ ఉంటుంది
- **రంగు** చిత్రంలో మూడు పొరలు (ఛానెల్స్) ఉంటాయి: ఎరుపు, ఆకుపచ్చ, మరియు నీలం — ఒకదానిపై ఒకటి అమర్చబడి ఉంటాయి
- ఒక సాధారణ స్మార్ట్‌ఫోన్ ఫోటో (12 MP) లో **12 మిలియన్ పిక్సెల్స్** ఉంటాయి — రంగు చిత్రానికి అది 36 మిలియన్ సంఖ్యలు!

```python
# Loading an image and viewing its pixel values
from PIL import Image
import numpy as np

img = Image.open("dog.jpg")
pixels = np.array(img)

print(f"Image shape: {pixels.shape}")
# Output example: (480, 640, 3) → 480 rows, 640 columns, 3 colour channels

print(f"Total pixel values: {pixels.size:,}")
# Output example: 921,600

# A single pixel in the top-left corner
print(f"Top-left pixel (R, G, B): {pixels[0, 0]}")
# Output example: [142, 178, 225] — a light blue sky pixel
```

### పిక్సెల్స్ నుండి ఫీచర్‌ల వరకు

ముడి పిక్సెల్ విలువలు వాటంతట అవే అర్థరహితం. AI కి **ఫీచర్‌లు** — అర్థవంతమైన నమూనాలు — గుర్తించడం అవసరం:
- **తక్కువ-స్థాయి ఫీచర్‌లు:** అంచులు, మూలలు, రంగు ప్రవణతలు
- **మధ్య-స్థాయి ఫీచర్‌లు:** టెక్చర్‌లు, ఆకారాలు, భాగాలు (కళ్ళు, చక్రాలు, కిటికీలు)
- **ఉన్నత-స్థాయి ఫీచర్‌లు:** మొత్తం వస్తువులు (ముఖం, కారు, భవనం)

ఇది చదవడం నేర్చుకోవడం లాంటిదే: మొదట మీరు అక్షరాలు (తక్కువ-స్థాయి), తర్వాత పదాలు (మధ్య-స్థాయి), తర్వాత వాక్యాలు మరియు అర్థం (ఉన్నత-స్థాయి) నేర్చుకుంటారు.

<FunFact>మానవ విజువల్ కార్టెక్స్ కూడా చిత్రాలను పొరల్లో ప్రాసెస్ చేస్తుంది! మీ కళ్ళ దగ్గరి న్యూరాన్లు సాధారణ అంచులు మరియు రంగులను గుర్తిస్తాయి, అదే సమయంలో మెదడులోని లోతైన ప్రాంతాలు సంక్లిష్ట వస్తువులు మరియు ముఖాలను గుర్తిస్తాయి. CNNలు అక్షరాలా ఈ జీవసంబంధ నిర్మాణం నుండి ప్రేరణ పొందాయి.
</FunFact>

## కన్వల్యూషనల్ న్యూరల్ నెట్‌వర్క్‌లు (CNNs) 🧠

**కన్వల్యూషనల్ న్యూరల్ నెట్‌వర్క్** కంప్యూటర్ విజన్ యొక్క ముఖ్య స్తంభం. ఇది ఎలా పనిచేస్తుందో అర్థం చేసుకుందాం — భారీ గణితం అవసరం లేదు.

### ముఖ్య ఆలోచన: స్లైడింగ్ ఫిల్టర్‌లు
ఒక చిత్రంపై చిన్న భూతద్దం (ఉదాహరణకు 3×3 పిక్సెల్స్) ఉంచి ప్రతి స్థానంలో జరుపుతున్నట్లు ఊహించుకోండి. ప్రతి స్థానంలో, ఫిల్టర్ ఒక నిర్దిష్ట నమూనా కోసం చూస్తుంది — నిలువు అంచు, అడ్డ గీత, వంపు. ప్రతి ఫిల్టర్ **ఫీచర్ మ్యాప్** అనే కొత్త చిత్రాన్ని ఉత్పత్తి చేస్తుంది, అది ఆ నమూనా ఎక్కడ కనుగొనబడిందో హైలైట్ చేస్తుంది.

### CNN పొరలు (దృశ్య అవగాహన)
| పొర | ఏమి చేస్తుంది | సారూప్యం |
|------|-------------|---------|
| **కన్వల్యూషనల్** | ఫీచర్‌లను గుర్తించడానికి చిత్రంపై ఫిల్టర్‌లను జరుపుతుంది | ఒక డిటెక్టివ్ భూతద్దంతో నేర స్థలంలో ప్రతి అంగుళాన్ని పరిశీలించడం |
| **యాక్టివేషన్ (ReLU)** | బలమైన సంకేతాలను మాత్రమే ఉంచుతుంది, శబ్దాన్ని తొలగిస్తుంది | కీలక ఆధారాలను హైలైట్ చేసి పరధ్యానాన్ని విస్మరించడం |
| **పూలింగ్** | ముఖ్యమైన సమాచారాన్ని నిలుపుకుంటూ ఫీచర్ మ్యాప్‌లను కుదిస్తుంది | ఒక అధ్యాయాన్ని బుల్లెట్ పాయింట్లుగా సంక్షేపించడం |
| **ఫుల్లీ కనెక్టెడ్** | తుది నిర్ణయం తీసుకోవడానికి అన్ని ఫీచర్‌లను కలుపుతుంది | జ్యూరీ చర్చించి తీర్పు ఇవ్వడం |

```python
# Building a simple CNN with TensorFlow/Keras
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense

model = Sequential([
    # First convolutional layer: 32 filters, 3x3 each
    Conv2D(32, (3, 3), activation='relu', input_shape=(64, 64, 3)),
    MaxPooling2D(pool_size=(2, 2)),

    # Second convolutional layer: 64 filters
    Conv2D(64, (3, 3), activation='relu'),
    MaxPooling2D(pool_size=(2, 2)),

    # Flatten and classify
    Flatten(),
    Dense(128, activation='relu'),
    Dense(10, activation='softmax'),  # 10 classes
])

model.summary()
```

<Callout>
మీరు ఫిల్టర్‌లను చేతితో రూపొందించాల్సిన అవసరం లేదు! శిక్షణ సమయంలో, CNN ఏ ఫిల్టర్‌లు ఉపయోగకరమైనవో **స్వయంగా నేర్చుకుంటుంది**. ప్రారంభ పొరలు అంచులను గుర్తించడం నేర్చుకోవచ్చు, లోతైన పొరలు చెవులు, కళ్ళు లేదా మొత్తం ముఖాలను గుర్తించడం నేర్చుకోవచ్చు — అంతా డేటా నుండి స్వయంచాలకంగా.
</Callout>

## కంప్యూటర్ విజన్ యొక్క మూడు ప్రధాన పనులు 🎯

### 1. ఇమేజ్ క్లాసిఫికేషన్
**ప్రశ్న:** "ఈ చిత్రంలో ఏముంది?"
మోడల్ మొత్తం చిత్రానికి ఒకే లేబుల్‌ను కేటాయిస్తుంది: "పిల్లి", "కారు", "పిజ్జా". మీరు AI Sprouts లో నేర్చుకున్నదే ఇది — మోడల్‌కు ఒక చిత్రం ఇవ్వండి, ఒక లేబుల్ తిరిగి పొందండి.

### 2. ఆబ్జెక్ట్ డిటెక్షన్
**ప్రశ్న:** "ఇక్కడ ఏ వస్తువులు ఉన్నాయి మరియు అవి ఎక్కడ ఉన్నాయి?"
మోడల్ ప్రతి వస్తువు చుట్టూ **బౌండింగ్ బాక్స్‌లు** గీసి వాటికి లేబుల్ వేస్తుంది. సెల్ఫ్-డ్రైవింగ్ కార్లు పాదచారులు, ట్రాఫిక్ సంకేతాలు మరియు ఇతర వాహనాలను గుర్తించడానికి దీన్నే ఉపయోగిస్తాయి.

### 3. ఇమేజ్ సెగ్మెంటేషన్
**ప్రశ్న:** "ప్రతి వస్తువుకు ఖచ్చితంగా ఏ పిక్సెల్స్ చెందుతాయి?"
మోడల్ ప్రతి పిక్సెల్‌ను దాని వస్తువు తరగతితో రంగు వేస్తుంది. ఇది పిక్సెల్-ఖచ్చిత రూపురేఖలను అందిస్తుంది — వైద్య ఇమేజింగ్ (కణితి సరిహద్దులు) మరియు AR ఫిల్టర్‌లకు (మీ ముఖాన్ని నేపథ్యం నుండి వేరు చేయడం) ఇది అత్యంత ముఖ్యం.

```python
# Using a pre-trained object detection model
# This uses YOLO (You Only Look Once) — one of the fastest detectors

# Pseudocode for clarity
def detect_objects(image_path):
    """Detect and label objects in an image."""
    model = load_pretrained_model("yolov8")
    image = load_image(image_path)

    results = model.predict(image)

    for detection in results:
        print(f"Object: {detection.label}")
        print(f"  Confidence: {detection.confidence:.1%}")
        print(f"  Location: {detection.bounding_box}")
        print()

# Example output:
# Object: person
#   Confidence: 97.2%
#   Location: (120, 50, 340, 480)
# Object: bicycle
#   Confidence: 89.5%
#   Location: (200, 300, 450, 520)
```

### 📱 AR ఫిల్టర్‌లు (Snapchat / Instagram)

- మొబైల్ ప్రాసెసర్‌లపై నడవడానికి ఆప్టిమైజ్ చేసిన తేలికపాటి మోడల్‌లను ఉపయోగిస్తుంది

## ప్రాక్టికల్: ముందుగా శిక్షణ పొందిన మోడల్ అంచనాలను అర్థం చేసుకోవడం 🔬

కంప్యూటర్ విజన్‌తో ప్రయోగించడానికి మీరు మొదటి నుండి మోడల్‌ను శిక్షణ ఇవ్వాల్సిన అవసరం లేదు. **ResNet**, **MobileNet** లేదా **EfficientNet** వంటి ముందుగా శిక్షణ పొందిన మోడల్‌లు ఇప్పటికే లక్షలాది చిత్రాల నుండి నేర్చుకున్నాయి.
```python
# Using a pre-trained model to classify an image
from tensorflow.keras.applications import MobileNetV2
from tensorflow.keras.applications.mobilenet_v2 import (
    preprocess_input, decode_predictions
)
from tensorflow.keras.preprocessing import image
import numpy as np

# Load pre-trained MobileNetV2 (trained on ImageNet — 1000 classes)
model = MobileNetV2(weights='imagenet')

# Load and preprocess an image
img = image.load_img("photo.jpg", target_size=(224, 224))
img_array = preprocess_input(
    np.expand_dims(image.img_to_array(img), axis=0)
)

# Get predictions
predictions = model.predict(img_array)
top_3 = decode_predictions(predictions, top=3)[0]

print("Top 3 Predictions:")
for rank, (id, label, confidence) in enumerate(top_3, 1):
    bar = "█" * int(confidence * 40)
    print(f"  {rank}. {label}: {confidence:.1%} {bar}")

# Example output:
#   1. golden_retriever: 92.3% █████████████████████████████████████
#   2. Labrador_retriever: 4.1% ██
#   3. cocker_spaniel: 1.8% █
```

### ఏమి అన్వేషించాలి
1. **వేర్వేరు చిత్రాలను ప్రయత్నించండి** — అసాధారణ కోణాలు లేదా లైటింగ్‌ను మోడల్ ఎలా నిర్వహిస్తుంది?
2. **ఎడ్జ్ కేస్‌లను పరీక్షించండి** — బ్లర్రీ చిత్రాలు, బొమ్మలు లేదా ఆప్టికల్ ఇల్యూజన్‌లతో ఏమవుతుంది?
3. **కాన్ఫిడెన్స్‌ను తనిఖీ చేయండి** — అధిక కాన్ఫిడెన్స్ అంటే ఎల్లప్పుడూ అంచనా సరైనదేనా?
4. **పక్షపాతాల కోసం చూడండి** — మోడల్ వైవిధ్యమైన విషయాలపై సమానంగా పనిచేస్తుందా?

<Callout>MobileNetV2 ప్రత్యేకంగా మొబైల్ ఫోన్‌లపై నడవడానికి రూపొందించబడింది. దీనిలో కేవలం 3.4 మిలియన్ పారామీటర్‌లు ఉన్నాయి — ResNet-152 యొక్క 60 మిలియన్‌లతో పోలిస్తే — ఆకట్టుకునే ఖచ్చితత్వాన్ని కొనసాగిస్తూ. "డెప్త్‌వైజ్ సెపరబుల్ కన్వల్యూషన్స్" అనే తెలివైన ఆర్కిటెక్చర్ ట్రిక్ ద్వారా ఇది సాధ్యమవుతుంది, ఇది గణనను తగ్గిస్తుంది.
</Callout>

## సంక్షిప్త పునశ్చరణ 🎯
1. కంప్యూటర్‌లు చిత్రాలను **సంఖ్యల గ్రిడ్‌లుగా** చూస్తాయి — ఎరుపు, ఆకుపచ్చ మరియు నీలం విలువలతో కూడిన పిక్సెల్స్
2. **CNNలు** అంచుల నుండి వస్తువుల వరకు ఫీచర్‌లను స్వయంచాలకంగా గుర్తించడానికి స్లైడింగ్ ఫిల్టర్‌లను ఉపయోగిస్తాయి
3. మూడు ప్రధాన పనులు **క్లాసిఫికేషన్** (ఏమిటి?), **డిటెక్షన్** (ఏమిటి మరియు ఎక్కడ?), మరియు **సెగ్మెంటేషన్** (పిక్సెల్-స్థాయి లేబుల్స్)
4. వాస్తవ అనువర్తనాలలో **సెల్ఫ్-డ్రైవింగ్ కార్లు**, **ఫేషియల్ రికగ్నిషన్**, **AR ఫిల్టర్‌లు** మరియు **నాణ్యత నియంత్రణ** ఉన్నాయి
5. MobileNetV2 వంటి **ముందుగా శిక్షణ పొందిన మోడల్‌లు** మొదటి నుండి శిక్షణ ఇవ్వకుండా ప్రయోగించడానికి అనుమతిస్తాయి
6. **నీతిశాస్త్రం** — పక్షపాతం, నిఘా, డీప్‌ఫేక్‌లు, మరియు గోప్యత — కంప్యూటర్ విజన్‌ను మనం ఎలా ఉపయోగిస్తామో మార్గనిర్దేశం చేయాలి
