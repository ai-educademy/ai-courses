# Kaggle పోటీ గైడ్ - Kaggle లో ఎలా పోటీ పడాలి (మరియు నేర్చుకోవాలి)

Kaggle కేవలం పోటీ ప్లాట్‌ఫారమ్ మాత్రమే కాదు - ఇది ప్రపంచంలోనే అతిపెద్ద applied ML తరగతి గది. 15 మిలియన్లకు పైగా data scientists నైపుణ్యాలు మెరుగుపరచడానికి, portfolios నిర్మించడానికి, మరియు ఉద్యోగాలు పొందడానికి దీన్ని ఉపయోగిస్తారు. ఈ పాఠం సమర్థవంతంగా పోటీ పడడం మరియు ప్రతి పోటీ నుండి గరిష్ట అభ్యసనాన్ని పొందడం నేర్పిస్తుంది.

Google, Meta, మరియు టాప్ startups లోని hiring managers Kaggle leaderboards నుండి చురుగ్గా నియమిస్తారు. గెలవకపోయినా, బాగా డాక్యుమెంట్ చేసిన notebooks తో బలమైన profile ఆచరణాత్మక సామర్థ్యాన్ని ప్రదర్శిస్తుంది.

## 🔍 EDA - మీ మొదటి 48 గంటలు

డేటాను అర్థం చేసుకోకుండా ఎప్పుడూ modelling మొదలుపెట్టకండి. క్రమశిక్షణతో కూడిన EDA workflow:

```python
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("train.csv")

# Shape మరియు types - మీరు ఏమి పనిచేస్తున్నారో తెలుసుకోండి
print(f"Shape: {df.shape}")
print(f"Missing values:\n{df.isnull().sum().sort_values(ascending=False).head(10)}")

# Target distribution - ఇది balanced గా ఉందా?
df["target"].value_counts(normalize=True).plot(kind="bar")
plt.title("Target Distribution")
plt.show()

# Correlations - త్వరిత signal కనుగొనండి
correlations = df.select_dtypes(include="number").corr()["target"].sort_values()
print(correlations.head(10))  # Strongest negative correlations
print(correlations.tail(10))  # Strongest positive correlations
```

**Leakage కోసం తనిఖీ చేయండి** - prediction సమయంలో ఉండని features. బిగినర్లు చేసే నంబర్ వన్ తప్పు ఇదే. ఒక feature target తో అనుమానాస్పదంగా బాగా correlate అయితే, సెలబ్రేట్ చేయడానికి ముందు విచారణ చేయండి.

## 🛠️ గెలిపించే Feature Engineering

టాప్ పోటీదారులు 70% సమయాన్ని features పై ఖర్చు చేస్తారు, model tuning పై కాదు. యుద్ధ-పరీక్షిత టెక్నిక్‌లు:

**Aggregation features** - వివిధ స్థాయిల్లో group statistics:
```python
for col in ["category", "store_id", "day_of_week"]:
    stats = df.groupby(col)["sales"].agg(["mean", "std", "median"])
    stats.columns = [f"{col}_sales_{s}" for s in ["mean", "std", "median"]]
    df = df.merge(stats, on=col, how="left")
```

**Target encoding** - categories ను smoothed target means తో భర్తీ చేయండి (leakage నివారించడానికి fold-based encoding ఉపయోగించండి).

**Lag features** - time series కోసం, మునుపటి విలువలు బంగారం:
```python
for lag in [1, 7, 14, 28]:
    df[f"sales_lag_{lag}"] = df.groupby("store_id")["sales"].shift(lag)
```

## ✅ Cross-Validation - మీ Local Score ను నమ్మండి

మీ local CV score public leaderboard కంటే ఎక్కువ ముఖ్యం. బలమైన validation వ్యూహం:

```python
from sklearn.model_selection import StratifiedKFold

skf = StratifiedKFold(n_splits=5, shuffle=True, random_state=42)
oof_predictions = np.zeros(len(X))

for fold, (train_idx, val_idx) in enumerate(skf.split(X, y)):
    X_train, X_val = X.iloc[train_idx], X.iloc[val_idx]
    y_train, y_val = y.iloc[train_idx], y.iloc[val_idx]

    model = lgb.LGBMClassifier(n_estimators=1000, learning_rate=0.05)
    model.fit(X_train, y_train, eval_set=[(X_val, y_val)],
              callbacks=[lgb.early_stopping(50)])
    oof_predictions[val_idx] = model.predict_proba(X_val)[:, 1]

print(f"OOF AUC: {roc_auc_score(y, oof_predictions):.5f}")
```

**బంగారు నియమం**: మీ local CV మరియు public LB భిన్నాభిప్రాయం చెబితే, మీ CV ను నమ్మండి. Public leaderboard test data లో కొంత భాగం మాత్రమే ఉపయోగిస్తుంది - దానికి overfit అవడం ఉచ్చు.

<ThinkAboutIt>మీరు public leaderboard లో 15వ స్థానంలో ఉన్నారు కానీ మీ local CV ప్రకారం మీ model overfit అయింది. చివరి submission కోసం మీ అత్యుత్తమ LB score సబ్మిట్ చేస్తారా లేదా అత్యుత్తమ CV score? ఎందుకు?</ThinkAboutIt>

## 🎭 Ensemble Methods - చివరి పుష్

దాదాపు ప్రతి విజయవంతమైన solution ensembles ఉపయోగిస్తుంది. మూడు కీలక టెక్నిక్‌లు:

**Bagging** - ఒకే model ను వేర్వేరు data subsets పై train చేసి, predictions ను average చేయండి. Variance తగ్గిస్తుంది.

**Stacking** - వైవిధ్యమైన base models నుండి out-of-fold predictions పై meta-model train చేయండి:
```python
# Level 1: వైవిధ్యమైన base models OOF predictions ఉత్పత్తి చేస్తాయి
base_preds = np.column_stack([lgbm_oof, xgb_oof, catboost_oof, nn_oof])

# Level 2: logistic regression సరైన combination నేర్చుకుంటుంది
from sklearn.linear_model import LogisticRegression
meta = LogisticRegression()
meta.fit(base_preds, y_train)
```

**Blending** - model predictions యొక్క weighted average. Stacking కంటే సరళం కానీ ప్రభావవంతం:
```python
final = 0.4 * lgbm_pred + 0.35 * xgb_pred + 0.25 * catboost_pred
```

## 📓 గొప్ప Kaggle Notebooks రాయడం

Public notebooks మీ ప్రతిష్ఠను నిర్మిస్తాయి. మెడల్-అర్హమైన notebook లో ఉండాల్సినవి:

1. **స్పష్టమైన కథనం** - కేవలం code కాదు, మీ ఆలోచన వివరించండి

3. **విజువలైజేషన్లు** - అంతర్దృష్టులను వెల్లడించే plots, కేవలం అలంకరణ కాదు
