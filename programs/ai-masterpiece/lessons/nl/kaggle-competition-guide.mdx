## 🔍 EDA - Je Eerste 48 Uur

Begin nooit met modelleren voordat je je data begrijpt. Een gedisciplineerde EDA-workflow:

```python
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("train.csv")

# Vorm en types - weet waarmee je werkt
print(f"Shape: {df.shape}")
print(f"Missing values:\n{df.isnull().sum().sort_values(ascending=False).head(10)}")

# Verdeling van de target - is deze gebalanceerd?
df["target"].value_counts(normalize=True).plot(kind="bar")
plt.title("Target Distribution")
plt.show()

# Correlaties - vind snel signaal
correlations = df.select_dtypes(include="number").corr()["target"].sort_values()
print(correlations.head(10))  # Sterkste negatieve correlaties
print(correlations.tail(10))  # Sterkste positieve correlaties
```

**Controleer op leakage** - features die niet zouden bestaan op het moment van voorspelling. Dit is de nummer-één fout van beginners. Als een feature verdacht sterk correleert met de target, onderzoek het eerst voordat je gaat vieren.

## 🛠️ Feature Engineering die Wint

Topkandidaten besteden 70% van hun tijd aan features, niet aan modeltuning. Beproefde technieken:

**Aggregatiefeatures** - groepsstatistieken op verschillende granulariteiten:
```python
for col in ["category", "store_id", "day_of_week"]:
    stats = df.groupby(col)["sales"].agg(["mean", "std", "median"])
    stats.columns = [f"{col}_sales_{s}" for s in ["mean", "std", "median"]]
    df = df.merge(stats, on=col, how="left")
```

**Target encoding** - vervang categorieën door afgevlakte target-gemiddelden (gebruik fold-gebaseerde encoding om leakage te voorkomen).

**Lag features** - voor tijdreeksen zijn eerdere waarden goud waard:
```python
for lag in [1, 7, 14, 28]:
    df[f"sales_lag_{lag}"] = df.groupby("store_id")["sales"].shift(lag)
```

## ✅ Cross-Validation - Vertrouw Je Lokale Score

Je lokale CV-score is belangrijker dan het publieke leaderboard. Een robuuste validatiestrategie:

```python
from sklearn.model_selection import StratifiedKFold

skf = StratifiedKFold(n_splits=5, shuffle=True, random_state=42)
oof_predictions = np.zeros(len(X))

for fold, (train_idx, val_idx) in enumerate(skf.split(X, y)):
    X_train, X_val = X.iloc[train_idx], X.iloc[val_idx]
    y_train, y_val = y.iloc[train_idx], y.iloc[val_idx]

    model = lgb.LGBMClassifier(n_estimators=1000, learning_rate=0.05)
    model.fit(X_train, y_train, eval_set=[(X_val, y_val)],
              callbacks=[lgb.early_stopping(50)])
    oof_predictions[val_idx] = model.predict_proba(X_val)[:, 1]

print(f"OOF AUC: {roc_auc_score(y, oof_predictions):.5f}")
```

**Gouden regel**: als je lokale CV en het publieke leaderboard het niet eens zijn, vertrouw je CV. Het publieke leaderboard gebruikt slechts een fractie van de testdata - erop overfitten is een valkuil.

## 🎭 Ensemble-methoden - De Laatste Push

Bijna elke winnende oplossing gebruikt ensembles. Drie kerntechnieken:

**Bagging** - train hetzelfde model op verschillende datasubsets, middel de voorspellingen. Vermindert variantie.

**Stacking** - train een metamodel op out-of-fold-voorspellingen van diverse basismodellen:
```python
# Niveau 1: diverse basismodellen produceren OOF-voorspellingen
base_preds = np.column_stack([lgbm_oof, xgb_oof, catboost_oof, nn_oof])

# Niveau 2: logistische regressie leert de optimale combinatie
from sklearn.linear_model import LogisticRegression
meta = LogisticRegression()
meta.fit(base_preds, y_train)
```

**Blending** - gewogen gemiddelde van modelvoorspellingen. Eenvoudiger dan stacking maar effectief:
```python
final = 0.4 * lgbm_pred + 0.35 * xgb_pred + 0.25 * catboost_pred
```

## 📓 Geweldige Kaggle Notebooks Schrijven

Publieke notebooks bouwen je reputatie op. Een medaillewaardige notebook bevat:

1. **Helder verhaal** - leg je redenering uit, niet alleen je code
