## Rappel sur les nombres binaires

Chaque entier est une séquence de bits. Le nombre 13 en binaire s'écrit `1101` :

```
Bit position:  3  2  1  0
Bit value:     1  1  0  1
Decimal:       8+ 4+ 0+ 1 = 13
```

Le bit manipulation opère directement sur ces bits à l'aide d'opérateurs bitwise - remplaçant souvent des boucles par une seule opération.

## Opérations bitwise fondamentales

| Opérateur | Symbole | Exemple (5 = 101, 3 = 011) | Résultat |
|-----------|---------|----------------------------|----------|
| AND       | `&`     | `101 & 011`                | `001` (1) |
| OR        | `\|`    | `101 \| 011`               | `111` (7) |
| XOR       | `^`     | `101 ^ 011`                | `110` (6) |
| NOT       | `~`     | `~101`                     | `...010` |
| Left shift  | `<<`  | `101 << 1`               | `1010` (10) |
| Right shift | `>>`  | `101 >> 1`               | `10` (2) |

## Astuces essentielles

### Vérifier si c'est une puissance de deux

Une puissance de 2 possède exactement un bit à 1 : `n & (n - 1) == 0`.

```
  n = 8:  1000          n = 6:  0110
  n-1=7:  0111          n-1=5:  0101
  &    :  0000 ✓ power  &    :  0100 ✗ not power
```

### Compter les bits à 1 (algorithme de Brian Kernighan)

Chaque `n & (n - 1)` supprime le bit à 1 le plus bas. On compte les itérations jusqu'à n = 0.

```python
def count_bits(n):
    count = 0
    while n:
        n &= n - 1
        count += 1
    return count
```

Cela s'exécute en O(k) où k est le nombre de bits à 1 - pas en O(32).

### Échanger sans variable temporaire

```python
a ^= b
b ^= a
a ^= b
```

Fonctionne car XOR est son propre inverse : `a ^ b ^ b = a`.

## Single Number - XOR Cancellation

Chaque élément apparaît deux fois sauf un. Trouvez-le. Puisque `a ^ a = 0` et `a ^ 0 = a`, appliquez XOR à tous les éléments - les paires s'annulent, laissant le nombre unique.

```python
def single_number(nums):
    result = 0
    for n in nums:
        result ^= n
    return result
```

Temps O(n), espace O(1). Aucune hash map nécessaire.

## Single Number III - Deux nombres uniques

Quand *deux* nombres sont uniques, appliquez XOR à tout pour obtenir `x = a ^ b`. Trouvez un bit à 1 dans `x` (les nombres diffèrent à cette position). Partitionnez tous les nombres par ce bit et appliquez XOR à chaque groupe - chaque groupe contient exactement un nombre unique.

```python
def single_number_iii(nums):
    xor = 0
    for n in nums:
        xor ^= n
    diff_bit = xor & (-xor)  # isolate rightmost set bit
    a = b = 0
    for n in nums:
        if n & diff_bit:
            a ^= n
        else:
            b ^= n
    return [a, b]
```

## Bit Masking pour les sous-ensembles

Un bitmask de longueur n représente un sous-ensemble de n éléments. Le bit `i` à 1 signifie que l'élément `i` est inclus. Itérer de `0` à `2^n - 1` génère tous les sous-ensembles.

```python
def all_subsets(items):
    n = len(items)
    for mask in range(1 << n):
        subset = [items[i] for i in range(n)
                  if mask & (1 << i)]
        yield subset
```

Cette technique convient aux problèmes où n ≤ 20 - assez petit pour une énumération en 2^n.

<FunFact>Les permissions de fichiers Linux (rwxr-xr-x = 755) sont un bitmask du monde réel. Chaque chiffre encode trois bits de permission : lecture (4), écriture (2), exécution (1), combinés avec OR.</FunFact>

## Inverser les bits (LeetCode 190)

Inversez un entier non signé 32 bits, bit par bit. Décalez le résultat à gauche et ajoutez par OR le bit le plus bas de l'entrée, puis décalez l'entrée à droite.

```python
def reverse_bits(n):
    result = 0
    for _ in range(32):
        result = (result << 1) | (n & 1)
        n >>= 1
    return result
```

## Hamming Distance (LeetCode 461)

Le nombre de positions où les bits correspondants diffèrent. Appliquez XOR aux deux nombres - les bits à 1 dans le résultat sont les positions différentes. Comptez-les avec l'astuce de Brian Kernighan.

```python
def hamming_distance(x, y):
    return count_bits(x ^ y)
```

## Applications pratiques

Le bit manipulation est omniprésent : les permissions Unix utilisent des masques OR/AND, les feature flags stockent des toggles dans des bits individuels, les masques de sous-réseau utilisent AND pour le routage, et la détection d'erreurs repose sur les checksums XOR.
