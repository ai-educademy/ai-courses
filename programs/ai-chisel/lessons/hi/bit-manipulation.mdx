## Binary Number रिव्यू

हर integer bits की एक sequence होता है। 13 को binary में लिखें तो `1101` मिलता है:

```
Bit position:  3  2  1  0
Bit value:     1  1  0  1
Decimal:       8+ 4+ 0+ 1 = 13
```

Bit manipulation इन bits पर सीधे bitwise operators से काम करता है — अक्सर loops की जगह एक ही operation काफ़ी होता है।

## Core Bitwise Operations

| Operator | Symbol | Example (5 = 101, 3 = 011) | Result |
|----------|--------|----------------------------|--------|
| AND      | `&`    | `101 & 011`                | `001` (1) |
| OR       | `\|`   | `101 \| 011`               | `111` (7) |
| XOR      | `^`    | `101 ^ 011`                | `110` (6) |
| NOT      | `~`    | `~101`                     | `...010` |
| Left shift  | `<<` | `101 << 1`              | `1010` (10) |
| Right shift | `>>` | `101 >> 1`              | `10` (2) |

## ज़रूरी Bit Tricks

### Power of Two चेक करना

Power of 2 में सिर्फ़ एक set bit होता है: `n & (n - 1) == 0`।

```
  n = 8:  1000          n = 6:  0110
  n-1=7:  0111          n-1=5:  0101
  &    :  0000 ✓ power  &    :  0100 ✗ not power
```

### Set Bits गिनना (Brian Kernighan's Algorithm)

हर `n & (n - 1)` सबसे नीचे वाला set bit हटा देता है। जब तक n = 0 न हो, iterations गिनते रहो।

```python
def count_bits(n):
    count = 0
    while n:
        n &= n - 1
        count += 1
    return count
```

यह O(k) में चलता है जहाँ k set bits की संख्या है — O(32) नहीं।

### बिना Temp Variable के Swap

```python
a ^= b
b ^= a
a ^= b
```

काम करता है क्योंकि XOR अपना ही inverse है: `a ^ b ^ b = a`।

## Single Number — XOR Cancellation

हर element दो बार आता है सिवाय एक के। उसे ढूँढो। चूँकि `a ^ a = 0` और `a ^ 0 = a`, सभी elements को XOR करो — pairs cancel हो जाएँगे, unique number बचेगा।

```python
def single_number(nums):
    result = 0
    for n in nums:
        result ^= n
    return result
```

Time O(n), space O(1)। कोई hash map नहीं चाहिए।

## Single Number III — दो Unique Numbers

जब *दो* numbers unique हों, तो सबको XOR करो — `x = a ^ b` मिलेगा। `x` में कोई भी set bit ढूँढो (वहाँ दोनों numbers अलग हैं)। उस bit से सभी numbers को दो groups में बाँटो और हर group को XOR करो — हर group में ठीक एक unique number होगा।

```python
def single_number_iii(nums):
    xor = 0
    for n in nums:
        xor ^= n
    diff_bit = xor & (-xor)  # isolate rightmost set bit
    a = b = 0
    for n in nums:
        if n & diff_bit:
            a ^= n
        else:
            b ^= n
    return [a, b]
```

<ThinkAboutIt>`xor & (-xor)` सबसे दाएँ set bit को isolate क्यों करता है? सोचो कि two's complement में negative numbers कैसे represent होते हैं।</ThinkAboutIt>

## Subsets के लिए Bit Masking

Length n का bitmask, n items का एक subset represent करता है। Bit `i` set होने का मतलब item `i` शामिल है। `0` से `2^n - 1` तक iterate करने पर सभी subsets generate हो जाते हैं।

```python
def all_subsets(items):
    n = len(items)
    for mask in range(1 << n):
        subset = [items[i] for i in range(n)
                  if mask & (1 << i)]
        yield subset
```

यह technique उन problems के लिए काम करती है जहाँ n ≤ 20 हो — 2^n enumeration के लिए काफ़ी छोटा।

<FunFact>Linux file permissions (rwxr-xr-x = 755) real-world bitmask का एक बढ़िया example हैं। हर digit तीन permission bits encode करती है: read (4), write (2), execute (1), जो OR से combine होती हैं।</FunFact>

## Bits Reverse करना (LeetCode 190)

एक 32-bit unsigned integer को bit by bit reverse करो। Result को left shift करो और input का lowest bit OR करो, फिर input को right shift करो।

```python
def reverse_bits(n):
    result = 0
    for _ in range(32):
        result = (result << 1) | (n & 1)
        n >>= 1
    return result
```

## Missing Number (LeetCode 268)

0 से n तक के n numbers की array जिसमें एक missing है। सभी array elements को indices 0 से n के साथ XOR करो — pairs cancel होंगे, missing number बचेगा। Sum formula भी चलेगा, लेकिन XOR में overflow का ख़तरा नहीं।

## Hamming Distance (LeetCode 461)

वो positions जहाँ corresponding bits अलग हैं, उनकी संख्या। दोनों numbers को XOR करो — result में set bits ही differing positions हैं। Brian Kernighan's trick से गिन लो।

```python
def hamming_distance(x, y):
    return count_bits(x ^ y)
```

## Practical Applications

Bit manipulation हर जगह दिखता है: Unix file permissions OR/AND masks इस्तेमाल करती हैं, feature flags individual bits में toggles store करते हैं, network subnet masks routing के लिए AND करते हैं, और error detection XOR checksums पर निर्भर करता है।
