## Interval-representatie

Een interval is een paar `[start, end]`. De meeste interval-problemen beginnen met één cruciale stap: **sorteer op starttijd**. Dit verandert een chaotische lijst in een reeks die je van links naar rechts kunt verwerken met eenvoudige vergelijkingen.

```
Unsorted: [3,6] [1,4] [8,10] [2,5]
Sorted:   [1,4] [2,5] [3,6] [8,10]
```

## Overlappende intervals mergen

Het fundamentele interval-probleem (LeetCode 56). Na het sorteren overlappen twee opeenvolgende intervals als de huidige start ≤ het vorige einde is.

```python
def merge(intervals):
    intervals.sort()
    merged = [intervals[0]]
    for start, end in intervals[1:]:
        if start <= merged[-1][1]:
            merged[-1][1] = max(merged[-1][1], end)
        else:
            merged.append([start, end])
    return merged
```

Laten we `[1,4] [2,5] [3,6] [8,10]` doorlopen:
- `[2,5]` overlapt met `[1,4]` → merge naar `[1,5]`
- `[3,6]` overlapt met `[1,5]` → merge naar `[1,6]`
- `[8,10]` geen overlap → nieuw interval

Resultaat: `[1,6] [8,10]`

## Insert Interval

Gegeven gesorteerde niet-overlappende intervals en een nieuw interval: voeg het in en merge waar nodig (LeetCode 57). Verdeel in drie fasen:

1. Voeg alle intervals toe die eindigen vóór het begin van het nieuwe
2. Merge alle overlappende intervals met het nieuwe
3. Voeg de resterende intervals ongewijzigd toe

## Meeting Rooms I — Kun je alles bijwonen?

Sorteer op starttijd. Als de start van een interval vóór het einde van het vorige valt, is er een conflict.

```python
def can_attend(intervals):
    intervals.sort()
    for i in range(1, len(intervals)):
        if intervals[i][0] < intervals[i - 1][1]:
            return False
    return True
```

## Meeting Rooms II — Minimum aantal zalen

Hier schittert de **event-based aanpak**. Scheid starts en einden in twee gesorteerde lijsten en sweep:

```python
import heapq

def min_rooms(intervals):
    intervals.sort()
    heap = []
    for start, end in intervals:
        if heap and heap[0] <= start:
            heapq.heappop(heap)
        heapq.heappush(heap, end)
    return len(heap)
```

De heap volgt de meeting die het vroegst eindigt. Als die eindigt vóór de volgende begint, hergebruiken we de zaal. De maximale grootte van de heap is het antwoord.

## Non-Overlapping Intervals — Minimum verwijderingen

Gegeven intervals, vind het minimum aantal te verwijderen zodat er geen overlap is (LeetCode 435). Greedy: sorteer op **eindtijd**, behoud altijd het interval dat het vroegst eindigt — dit laat de meeste ruimte voor toekomstige intervals.

```python
def erase_overlap(intervals):
    intervals.sort(key=lambda x: x[1])
    count, prev_end = 0, float('-inf')
    for start, end in intervals:
        if start >= prev_end:
            prev_end = end
        else:
            count += 1
    return count
```

## Interval List Intersections

Gegeven twee gesorteerde interval-lijsten, vind alle intersecties (LeetCode 986). Gebruik twee pointers — één per lijst. De intersectie van `[a1, a2]` en `[b1, b2]` is `[max(a1, b1), min(a2, b2)]` als deze geldig is (start ≤ einde). Verschuif de pointer waarvan het interval het eerst eindigt.

## De Sweep Line-techniek

In plaats van intervals paarsgewijs te vergelijken, ontleed elk interval in events:

```
[1,4] → +1 at 1, -1 at 4
[2,6] → +1 at 2, -1 at 6
```

Sorteer events op tijd. Sweep van links naar rechts met een lopende telling. De piektelling beantwoordt vragen over "maximale overlap". Dit generaliseert Meeting Rooms II naar willekeurige event-processing problemen.

**My Calendar I** (LeetCode 729): beheer geboekte intervals en wijs overlappingen af. **My Calendar II**: sta enkele overlappingen toe maar wijs triple-bookings af. De sweep line maakt meerlaagse overlapdetectie eenvoudig.
