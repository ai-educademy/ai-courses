## Binaire getallen opfrissen

Elk geheel getal is een reeks bits. Het getal 13 in binair is `1101`:

```
Bit position:  3  2  1  0
Bit value:     1  1  0  1
Decimal:       8+ 4+ 0+ 1 = 13
```

Bit manipulation werkt rechtstreeks op deze bits met bitwise operators - vaak worden lussen vervangen door een enkele operatie.

## Fundamentele bitwise operaties

| Operator | Symbool | Voorbeeld (5 = 101, 3 = 011) | Resultaat |
|----------|---------|-------------------------------|-----------|
| AND      | `&`     | `101 & 011`                   | `001` (1) |
| OR       | `\|`    | `101 \| 011`                  | `111` (7) |
| XOR      | `^`     | `101 ^ 011`                   | `110` (6) |
| NOT      | `~`     | `~101`                        | `...010` |
| Left shift  | `<<` | `101 << 1`                  | `1010` (10) |
| Right shift | `>>` | `101 >> 1`                  | `10` (2) |

## Essentiële bit-trucs

### Controleren of het een macht van twee is

Een macht van 2 heeft precies één gezette bit: `n & (n - 1) == 0`.

```
  n = 8:  1000          n = 6:  0110
  n-1=7:  0111          n-1=5:  0101
  &    :  0000 ✓ power  &    :  0100 ✗ not power
```

### Gezette bits tellen (Brian Kernighan's algoritme)

Elke `n & (n - 1)` verwijdert de laagste gezette bit. Tel de iteraties totdat n = 0.

```python
def count_bits(n):
    count = 0
    while n:
        n &= n - 1
        count += 1
    return count
```

Dit draait in O(k) waarbij k het aantal gezette bits is - niet O(32).

### Omwisselen zonder tijdelijke variabele

```python
a ^= b
b ^= a
a ^= b
```

Dit werkt omdat XOR zijn eigen inverse is: `a ^ b ^ b = a`.

## Single Number - XOR Cancellation

Elk element komt twee keer voor behalve één. Vind het. Aangezien `a ^ a = 0` en `a ^ 0 = a`, XOR je alle elementen - paren heffen elkaar op en het unieke getal blijft over.

```python
def single_number(nums):
    result = 0
    for n in nums:
        result ^= n
    return result
```

Tijd O(n), ruimte O(1). Geen hash map nodig.

## Single Number III - Twee unieke getallen

Wanneer *twee* getallen uniek zijn, XOR je alles om `x = a ^ b` te krijgen. Zoek een gezette bit in `x` (de getallen verschillen daar). Verdeel alle getallen op basis van die bit en XOR elke groep - elke groep bevat precies één uniek getal.

```python
def single_number_iii(nums):
    xor = 0
    for n in nums:
        xor ^= n
    diff_bit = xor & (-xor)  # isolate rightmost set bit
    a = b = 0
    for n in nums:
        if n & diff_bit:
            a ^= n
        else:
            b ^= n
    return [a, b]
```

<ThinkAboutIt>Waarom isoleert `xor & (-xor)` de meest rechtse gezette bit? Denk na over hoe negatieve getallen worden weergegeven in two's complement.</ThinkAboutIt>

## Bit Masking voor subsets

Een bitmask van lengte n vertegenwoordigt een subset van n items. Bit `i` gezet betekent dat item `i` is opgenomen. Itereren van `0` tot `2^n - 1` genereert alle subsets.

```python
def all_subsets(items):
    n = len(items)
    for mask in range(1 << n):
        subset = [items[i] for i in range(n)
                  if mask & (1 << i)]
        yield subset
```

Deze techniek is geschikt voor problemen waar n ≤ 20 - klein genoeg voor 2^n enumeratie.

<FunFact>Linux-bestandsrechten (rwxr-xr-x = 755) zijn een bitmask uit de praktijk. Elk cijfer codeert drie permission bits: lezen (4), schrijven (2), uitvoeren (1), gecombineerd met OR.</FunFact>

## Bits omdraaien (LeetCode 190)

Draai een 32-bit unsigned integer bit voor bit om. Shift het resultaat naar links en OR de laagste bit van de invoer erin, shift daarna de invoer naar rechts.

```python
def reverse_bits(n):
    result = 0
    for _ in range(32):
        result = (result << 1) | (n & 1)
        n >>= 1
    return result
```

## Missing Number (LeetCode 268)

Een array van n getallen van 0 tot n met één ontbrekend getal. XOR alle array-elementen met alle indices van 0 tot n - paren heffen elkaar op en het ontbrekende getal blijft over. De somformule werkt ook, maar XOR vermijdt mogelijke overflow.

## Hamming Distance (LeetCode 461)

Het aantal posities waar de corresponderende bits verschillen. XOR de twee getallen - gezette bits in het resultaat zijn de afwijkende posities. Tel ze met Brian Kernighan's trick.

```python
def hamming_distance(x, y):
    return count_bits(x ^ y)
```

## Praktische toepassingen

Bit manipulation is overal: Unix-bestandsrechten gebruiken OR/AND masks, feature flags slaan toggles op als individuele bits, netwerk-subnetmaskers gebruiken AND voor routing, en foutdetectie is gebaseerd op XOR checksums.
