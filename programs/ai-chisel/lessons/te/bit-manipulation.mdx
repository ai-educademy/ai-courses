## Binary Number సమీక్ష

ప్రతి integer bits యొక్క ఒక sequence. 13 అనే సంఖ్య binary లో `1101`:

```
Bit position:  3  2  1  0
Bit value:     1  1  0  1
Decimal:       8+ 4+ 0+ 1 = 13
```

Bit manipulation ఈ bits పై నేరుగా bitwise operators ఉపయోగించి పని చేస్తుంది — తరచుగా loops స్థానంలో ఒకే operation సరిపోతుంది.

## ప్రాథమిక Bitwise Operations

| Operator | Symbol | ఉదాహరణ (5 = 101, 3 = 011) | ఫలితం |
|----------|--------|----------------------------|--------|
| AND      | `&`    | `101 & 011`                | `001` (1) |
| OR       | `\|`   | `101 \| 011`               | `111` (7) |
| XOR      | `^`    | `101 ^ 011`                | `110` (6) |
| NOT      | `~`    | `~101`                     | `...010` |
| Left shift  | `<<` | `101 << 1`              | `1010` (10) |
| Right shift | `>>` | `101 >> 1`              | `10` (2) |

## అవసరమైన Bit Tricks

### Power of Two చెక్ చేయడం

Power of 2 లో సరిగ్గా ఒక్క set bit ఉంటుంది: `n & (n - 1) == 0`.

```
  n = 8:  1000          n = 6:  0110
  n-1=7:  0111          n-1=5:  0101
  &    :  0000 ✓ power  &    :  0100 ✗ not power
```

### Set Bits లెక్కించడం (Brian Kernighan's Algorithm)

ప్రతి `n & (n - 1)` అత్యంత తక్కువ set bit ని తొలగిస్తుంది. n = 0 అయ్యే వరకు iterations లెక్కించండి.

```python
def count_bits(n):
    count = 0
    while n:
        n &= n - 1
        count += 1
    return count
```

ఇది O(k) లో నడుస్తుంది, ఇక్కడ k అనేది set bits సంఖ్య — O(32) కాదు.

### Temp Variable లేకుండా Swap

```python
a ^= b
b ^= a
a ^= b
```

XOR తన స్వంత inverse కాబట్టి ఇది పని చేస్తుంది: `a ^ b ^ b = a`.

## Single Number — XOR Cancellation

ప్రతి element రెండు సార్లు వస్తుంది ఒక్కటి తప్ప. దాన్ని కనుగొనండి. `a ^ a = 0` మరియు `a ^ 0 = a` కాబట్టి, అన్ని elements ని XOR చేయండి — pairs cancel అవుతాయి, unique number మిగులుతుంది.

```python
def single_number(nums):
    result = 0
    for n in nums:
        result ^= n
    return result
```

Time O(n), space O(1). Hash map అవసరం లేదు.

## Single Number III — రెండు Unique Numbers

*రెండు* numbers unique అయినప్పుడు, అన్నింటినీ XOR చేస్తే `x = a ^ b` వస్తుంది. `x` లో ఏదైనా set bit కనుగొనండి (అక్కడ రెండు numbers భిన్నంగా ఉంటాయి). ఆ bit ఆధారంగా అన్ని numbers ని రెండు groups గా విభజించి, ప్రతి group ని XOR చేయండి — ప్రతి group లో సరిగ్గా ఒక unique number ఉంటుంది.

```python
def single_number_iii(nums):
    xor = 0
    for n in nums:
        xor ^= n
    diff_bit = xor & (-xor)  # isolate rightmost set bit
    a = b = 0
    for n in nums:
        if n & diff_bit:
            a ^= n
        else:
            b ^= n
    return [a, b]
```

<ThinkAboutIt>`xor & (-xor)` అత్యంత కుడివైపు set bit ని ఎందుకు isolate చేస్తుంది? Two's complement లో negative numbers ఎలా represent అవుతాయో ఆలోచించండి.</ThinkAboutIt>

## Subsets కోసం Bit Masking

n పొడవు గల bitmask, n items యొక్క ఒక subset ని represent చేస్తుంది. Bit `i` set అయి ఉంటే item `i` చేర్చబడింది. `0` నుండి `2^n - 1` వరకు iterate చేస్తే అన్ని subsets generate అవుతాయి.

```python
def all_subsets(items):
    n = len(items)
    for mask in range(1 << n):
        subset = [items[i] for i in range(n)
                  if mask & (1 << i)]
        yield subset
```

n ≤ 20 ఉన్న problems కోసం ఈ technique పని చేస్తుంది — 2^n enumeration కి సరిపోయేంత చిన్నది.

<FunFact>Linux file permissions (rwxr-xr-x = 755) ఒక real-world bitmask ఉదాహరణ. ప్రతి digit మూడు permission bits ని encode చేస్తుంది: read (4), write (2), execute (1), OR తో combine చేయబడతాయి.</FunFact>

## Bits Reverse చేయడం (LeetCode 190)

ఒక 32-bit unsigned integer ని bit వారీగా reverse చేయండి. Result ని left shift చేసి input యొక్క lowest bit ని OR చేయండి, తర్వాత input ని right shift చేయండి.

```python
def reverse_bits(n):
    result = 0
    for _ in range(32):
        result = (result << 1) | (n & 1)
        n >>= 1
    return result
```

## Missing Number (LeetCode 268)

0 నుండి n వరకు n numbers ఉన్న array లో ఒకటి missing. అన్ని array elements ని 0 నుండి n వరకు indices తో XOR చేయండి — pairs cancel అవుతాయి, missing number మిగులుతుంది. Sum formula కూడా పని చేస్తుంది, కానీ XOR overflow ప్రమాదాన్ని నివారిస్తుంది.

## Hamming Distance (LeetCode 461)

సంబంధిత bits భిన్నంగా ఉన్న positions సంఖ్య. రెండు numbers ని XOR చేయండి — result లో set bits భిన్నమైన positions. Brian Kernighan's trick తో లెక్కించండి.

```python
def hamming_distance(x, y):
    return count_bits(x ^ y)
```
