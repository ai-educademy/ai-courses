## Interval Representation

ఒక interval అంటే `[start, end]` జత. చాలా interval problems ఒక critical step తో మొదలవుతాయి: **start time ప్రకారం sort చేయడం**. ఇది ఒక అస్తవ్యస్తమైన list ను ఎడమ నుండి కుడికి simple comparisons తో process చేయగల sequence గా మారుస్తుంది.

```
Unsorted: [3,6] [1,4] [8,10] [2,5]
Sorted:   [1,4] [2,5] [3,6] [8,10]
```

## Overlapping Intervals ను Merge చేయడం

ఇది foundational interval problem (LeetCode 56). Sort చేసిన తర్వాత, current start ≤ previous end అయితే రెండు consecutive intervals overlap అవుతాయి.

```python
def merge(intervals):
    intervals.sort()
    merged = [intervals[0]]
    for start, end in intervals[1:]:
        if start <= merged[-1][1]:
            merged[-1][1] = max(merged[-1][1], end)
        else:
            merged.append([start, end])
    return merged
```

`[1,4] [2,5] [3,6] [8,10]` ను చూద్దాం:
- `[2,5]` `[1,4]` తో overlap అవుతుంది → `[1,5]` గా merge
- `[3,6]` `[1,5]` తో overlap అవుతుంది → `[1,6]` గా merge
- `[8,10]` overlap లేదు → కొత్త interval

ఫలితం: `[1,6] [8,10]`

## Insert Interval

Sorted non-overlapping intervals మరియు ఒక కొత్త interval ఇవ్వబడ్డాయి, దాన్ని insert చేసి అవసరమైతే merge చేయండి (LeetCode 57). మూడు phases గా విభజించండి:

1. కొత్త interval మొదలయ్యే ముందే అయిపోయే intervals అన్నీ add చేయండి
2. కొత్త interval తో overlap అయ్యే intervals అన్నింటినీ merge చేయండి
3. మిగిలిన intervals ను మార్పులు లేకుండా add చేయండి

<ThinkAboutIt>Insert Interval తరచుగా merge step స్పష్టంగా కనిపించకుండా present చేయబడుతుంది. Interview లో దీన్ని merge problem గా మీరు ఎలా గుర్తిస్తారు?</ThinkAboutIt>

## Meeting Rooms I - అన్నీ Attend చేయగలరా?

Start time ప్రకారం sort చేయండి. ఏదైనా interval యొక్క start మునుపటి interval యొక్క end కంటే ముందు ఉంటే, conflict ఉంది.

```python
def can_attend(intervals):
    intervals.sort()
    for i in range(1, len(intervals)):
        if intervals[i][0] < intervals[i - 1][1]:
            return False
    return True
```

## Meeting Rooms II - కనీసం ఎన్ని Rooms కావాలి

ఇక్కడ **event-based approach** ప్రకాశిస్తుంది. Starts మరియు ends ను రెండు sorted lists గా విడదీయండి, తర్వాత sweep చేయండి:

```python
import heapq

def min_rooms(intervals):
    intervals.sort()
    heap = []
    for start, end in intervals:
        if heap and heap[0] <= start:
            heapq.heappop(heap)
        heapq.heappush(heap, end)
    return len(heap)
```

Heap అత్యంత త్వరగా అయిపోయే meeting ను track చేస్తుంది. అది తదుపరి meeting మొదలయ్యే ముందు అయిపోతే, మనం room ను reuse చేస్తాము. Heap యొక్క maximum sizeే answer.

## Non-Overlapping Intervals - కనీసం ఎన్ని Remove చేయాలి

Intervals ఇవ్వబడ్డాయి, ఏ overlap లేకుండా ఉండేందుకు కనీసం ఎన్ని remove చేయాలో కనుగొనండి (LeetCode 435). Greedy: **end time** ప్రకారం sort చేయండి, ఎల్లప్పుడూ అత్యంత త్వరగా అయిపోయే interval ను ఉంచండి - ఇది భవిష్యత్ intervals కోసం ఎక్కువ స్థలం మిగులుస్తుంది.

```python
def erase_overlap(intervals):
    intervals.sort(key=lambda x: x[1])
    count, prev_end = 0, float('-inf')
    for start, end in intervals:
        if start >= prev_end:
            prev_end = end
        else:
            count += 1
    return count
```

## Interval List Intersections

రెండు sorted interval lists ఇవ్వబడ్డాయి, అన్ని intersections కనుగొనండి (LeetCode 986). రెండు pointers ఉపయోగించండి - ప్రతి list కు ఒకటి. `[a1, a2]` మరియు `[b1, b2]` యొక్క intersection `[max(a1, b1), min(a2, b2)]` - ఇది valid అయితే (start ≤ end). ఏ pointer యొక్క interval ముందు అయిపోతుందో దాన్ని ముందుకు జరపండి.

## Sweep Line Technique

Intervals ను pairwise compare చేయడానికి బదులు, ప్రతి interval ను events గా విడగొట్టండి:

```
[1,4] → +1 at 1, -1 at 4
[2,6] → +1 at 2, -1 at 6
```

Events ను time ప్రకారం sort చేయండి. ఎడమ నుండి కుడికి sweep చేయండి, running count maintain చేస్తూ. Peak count "maximum overlap" ప్రశ్నలకు answer ఇస్తుంది. ఇది Meeting Rooms II ను arbitrary event-processing problems కు generalise చేస్తుంది.

## Calendar Booking

**My Calendar I** (LeetCode 729): booked intervals maintain చేయండి మరియు overlaps reject చేయండి. **My Calendar II**: single overlaps allow చేయండి కానీ triple-bookings reject చేయండి. Sweep line multi-layer overlap detection ను straightforward గా చేస్తుంది.
