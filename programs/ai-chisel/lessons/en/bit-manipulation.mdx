## Binary Number Review

Every integer is a sequence of bits. The number 13 in binary is `1101`:

```
Bit position:  3  2  1  0
Bit value:     1  1  0  1
Decimal:       8+ 4+ 0+ 1 = 13
```

Bit manipulation operates directly on these bits using bitwise operators — often replacing loops with single operations.

## Core Bitwise Operations

| Operator | Symbol | Example (5 = 101, 3 = 011) | Result |
|----------|--------|----------------------------|--------|
| AND      | `&`    | `101 & 011`                | `001` (1) |
| OR       | `\|`   | `101 \| 011`               | `111` (7) |
| XOR      | `^`    | `101 ^ 011`                | `110` (6) |
| NOT      | `~`    | `~101`                     | `...010` |
| Left shift  | `<<` | `101 << 1`              | `1010` (10) |
| Right shift | `>>` | `101 >> 1`              | `10` (2) |

## Essential Bit Tricks

### Check if Power of Two

A power of 2 has exactly one set bit: `n & (n - 1) == 0`.

```
  n = 8:  1000          n = 6:  0110
  n-1=7:  0111          n-1=5:  0101
  &    :  0000 ✓ power  &    :  0100 ✗ not power
```

### Count Set Bits (Brian Kernighan's Algorithm)

Each `n & (n - 1)` removes the lowest set bit. Count iterations until n = 0.

```python
def count_bits(n):
    count = 0
    while n:
        n &= n - 1
        count += 1
    return count
```

This runs in O(k) where k is the number of set bits — not O(32).

### Swap Without Temp Variable

```python
a ^= b
b ^= a
a ^= b
```

Works because XOR is its own inverse: `a ^ b ^ b = a`.

## Single Number — XOR Cancellation

Every element appears twice except one. Find it. Since `a ^ a = 0` and `a ^ 0 = a`, XOR all elements — pairs cancel, leaving the unique number.

```python
def single_number(nums):
    result = 0
    for n in nums:
        result ^= n
    return result
```

Time O(n), space O(1). No hash map needed.

## Single Number III — Two Unique Numbers

When *two* numbers are unique, XOR everything to get `x = a ^ b`. Find any set bit in `x` (the numbers differ there). Partition all numbers by that bit and XOR each group — each group contains exactly one unique number.

```python
def single_number_iii(nums):
    xor = 0
    for n in nums:
        xor ^= n
    diff_bit = xor & (-xor)  # isolate rightmost set bit
    a = b = 0
    for n in nums:
        if n & diff_bit:
            a ^= n
        else:
            b ^= n
    return [a, b]
```

<ThinkAboutIt>Why does `xor & (-xor)` isolate the rightmost set bit? Think about how negative numbers are represented in two's complement.</ThinkAboutIt>

## Bit Masking for Subsets

A bitmask of length n represents a subset of n items. Bit `i` being set means item `i` is included. Iterating from `0` to `2^n - 1` generates all subsets.

```python
def all_subsets(items):
    n = len(items)
    for mask in range(1 << n):
        subset = [items[i] for i in range(n)
                  if mask & (1 << i)]
        yield subset
```

This technique powers solutions for problems with n ≤ 20 — small enough for 2^n enumeration.

<FunFact>Linux file permissions (rwxr-xr-x = 755) are a real-world bitmask. Each digit encodes three permission bits: read (4), write (2), execute (1), combined with OR.</FunFact>

## Reversing Bits (LeetCode 190)

Reverse a 32-bit unsigned integer bit by bit. Shift the result left and OR in the lowest bit of the input, then shift the input right.

```python
def reverse_bits(n):
    result = 0
    for _ in range(32):
        result = (result << 1) | (n & 1)
        n >>= 1
    return result
```

## Missing Number (LeetCode 268)

Array of n numbers from 0 to n with one missing. XOR all array elements with all indices 0 to n — pairs cancel, leaving the missing number. Alternatively, use the sum formula, but XOR avoids potential overflow.

## Hamming Distance (LeetCode 461)

The number of positions where corresponding bits differ. XOR the two numbers — set bits in the result are the differing positions. Count them with Brian Kernighan's trick.

```python
def hamming_distance(x, y):
    return count_bits(x ^ y)
```

## Practical Applications

Bit manipulation appears everywhere: Unix file permissions use OR/AND masks, feature flags store toggles as individual bits, network subnet masks use AND for routing, and error detection relies on XOR checksums.
