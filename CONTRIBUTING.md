# Contributing to AI Free Courses

Thank you for helping make AI education accessible to everyone! 🎉

## What you can contribute

### 📝 Content improvements
- Fix typos, grammar, or unclear explanations
- Add better examples or analogies
- Improve lesson flow and structure

### 🌐 Translations
- Translate lessons into new languages
- Review existing translations for accuracy

### 🧠 New content
- Propose new lessons (open an issue first)
- Add quizzes or exercises
- Create visual aids or diagrams

## Lesson format

Each lesson is a Markdown file in `programs/<program>/lessons/<lesson>.md` with frontmatter:

```yaml
---
title: "Lesson Title"
description: "Brief description"
order: 1
duration: "15 min"
---

# Lesson Title

Your content here...
```

## Guidelines

1. **Keep it simple** — explain concepts like you're teaching a friend
2. **Use examples** — real-world analogies help concepts stick
3. **Be inclusive** — avoid jargon without explanation
4. **Stay accurate** — cite sources for claims about AI capabilities

## Process

1. Fork the repo
2. Create a branch: `git checkout -b add-lesson-name`
3. Make your changes
4. Submit a pull request with a clear description

We review PRs within a few days. Thank you for contributing! 🙏
